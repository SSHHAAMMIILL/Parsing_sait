        Описание продукта

        The MultiStar Razor 40A Tower F4 is an excellent way of combining an F4 Flight Controller with 40A ESCs. This tower helps keep your drone or multirotor balanced as well as cutting down on ugly wiring. The boards come with standard 30.5x30.5mm mounting holes for easy installation in most frames.

         

        At the heart is the F4 Flight Controller with an STM32F405 MCU for quick processing speed and an MPU-6000 SPI Gyro for a quick response. Couple this with Betaflight firmware for easy setup and configuration. The FC comes with integrated OSD and microSD card reader for saving flight and black box data. There are easy plug connections (cables provided) for connections to different types of receivers (SBUS, PPM, Spektrum), FPV camera, video transmitter, buzzer or LEDs with 3.3V, 5V or 12V power.

         

        Connected directly to the Flight Controller is the ESC and Power Distribution Board with BLHeli_S firmware for superior throttle response. It supports Oneshot125, Oneshot42 and Multishot as well as Dshot150, Dshot300 and Dshot600. It also has a built-in current meter which displays directly onto the OSD. The 40A ESCs will provide all the power needed for most 170-450 multi-rotors.

         

        This is an outstanding package for any drone or multirotor user and will get you set up and be flying in no time.

         

        Features:
        Flight Controller 
        • MCU: STM32F405
        • Gyro: MPU-6000 SPI
        • FC firmware: Betaflight_OMNIBUSF4S
        • F4 flight controller, supports PPM, SBUS, SPEKTRUM1024/2048 etc. remote control / receiving mode
        • Supports max 32Gb TF-card which can record and save more flight/black box data
        • FC integrated OSD, users can adjust OSD parameter via BetaFlight configurator
        • FC integrated 3.3V,5V, 12V/ for VTX, camera, buzzer, LED and other peripheral devices
        • Provided several silicone cables for connectors on FC, and will give you an unprecedented experience for installation

        ESC
        • EFM8BB21F16G MCU, pipelined 8-bit C8051 core with 50 MHz maximum operating frequency
        • Dedicated high-frequency driver, makes the start smoother. ESC maximum speed is limited to 500k eRPM
        • ESC firmware: BLHeli-S, A-H-30
        • Two layers tower structure between 4in1 ESC and FC using quick pin connection. Two-way signal transmission makes the flight more stable
        • Built-in current meter, OSD directly display the real-time current without any additional connection
        • BLHeli-S firmware. It is designed for superior performance in multi-rotors and uses hardware generated motor PWM for smooth throttle response and silent operation
        • The code supports regular 1-2ms pulse width input, as well as Oneshot125, Oneshot42 and Multishot
        • Supports Dshot150, Dshot300 and Dshot600. Dshot is a digital signal, the anti-interference ability is stronger, and do not need throttle calibration

         

        Specs:

        Burst: 50A
